# classcreator

## SETUP

to setup run the following commands

```
git clone https://github.com/Moondog360/classcreator.git
cd ./classcreator
chmod +x ./setup
./setup
source ~/.zshrc
```

## RUNNING THE SCRIPT

to run the script just type the following command and replace <classname>. Example:
```
createclass test
ls *
Test.cpp Test.hpp
```
